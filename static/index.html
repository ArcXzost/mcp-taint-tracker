Detection Accuracy

Precision
—
Ratio of correctly detected true attacks to total alerts generated (1.0 = no false positives).
Recall
—
Ratio of correctly detected true attacks to total actual attacks in the benchmark (1.0 = no false negatives).
F1
—
Harmonic mean of precision and recall. A measure of overall detection accuracy.
FPR
—
False Positive Rate: Proportion of benign workflows incorrectly flagged as attacks.

Runtime Overhead

P50
—
Median latency overhead added by the taint tracking engine per intercepted event.
P95
—
95th percentile latency (worst-case tail latency) added per intercepted event.
Peak Mem
—
Maximum memory footprint consumed by attribution models and session graphs.
Attr Cost
—
Percentage of total processing time spent running attribution models vs logging.
